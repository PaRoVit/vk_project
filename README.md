# AI Image Enhancer (HDRnet)

Система улучшения изображений в браузере на основе нейросети **HDRnet** (Deep Bilateral Learning, SIGGRAPH 2017), работающей через TensorFlow.js. CNN анализирует уменьшенную копию изображения и предсказывает **bilateral grid** — сетку локально-аффинных цветовых преобразований, которые затем применяются к полноразмерному изображению через трилинейную интерполяцию в Web Worker.

## Возможности

- **Архитектура HDRnet** — двухпутевая CNN (local + global pathway) предсказывает bilateral grid 16×16×8 из 3×4 аффинных матриц, что позволяет делать **локальные, контент-зависимые** коррекции вместо плоских глобальных поправок.
- **Bilateral grid slicing** — сетка коэффициентов масштабируется до полного разрешения через трилинейную интерполяцию в пространстве (x, y, яркость) с естественным сохранением границ.
- **Гибридное предсказание** — результат CNN блендится с аналитическим алгоритмом (анализ гистограммы / цветовой статистики), поэтому система работает сразу без обучения.
- **Асинхронная обработка** — тяжёлые вычисления идут в Web Worker, браузер не блокируется. Прогресс обновляется в реальном времени.
- **Форматы** — JPG, PNG, HEIC, BMP.
- **Изображения до 15 Мпк** — CNN работает на миниатюре 256×256, полноразмерное изображение обрабатывается только лёгким циклом интерполяции.

## Архитектура

```
src/
  lib/
    ModelEngine.js     HDRnet CNN + аналитический алгоритм + композиция сетки
    ImageEnhancer.js   Публичный API (задачи, события, жизненный цикл)
    worker.js          Bilateral grid slicing (трилинейная интерполяция + аффинное преобразование)
    utils.js           Загрузка изображений, конвертация форматов
    index.js           Экспорты библиотеки
  main.js              Демо-интерфейс
  styles.css           Стили
index.html             Главная страница
```
